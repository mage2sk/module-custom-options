# Changelog

All notable changes to this extension are documented here. The format
is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] -- Initial release

### Added
- **Styled text inputs** -- `<input type="text">` and `<textarea>`
  with focus rings, placeholder text, and character-count notes.
- **File upload drop-zone** -- drag-and-drop area with file-type and
  dimension hints, replacing the plain `<input type="file">`.
- **Select / radio / checkbox styling** -- drop-down selects, card-style
  radio rows, and card-style checkbox rows with custom indicators.
- **Native HTML5 date & time pickers** -- uses `<input type="date">`,
  `<input type="time">`, and `<input type="datetime-local">` instead
  of legacy JS date-picker widgets.
- **CSS custom-property theming** -- every colour, border radius, and
  font is controlled via CSS variables defined in `theme-config.json`.
- **Admin toggle** -- `Stores -> Configuration -> Panth Extensions ->
  Custom Options -> Enable Custom Options Styling` (default: Yes).
- **Tax-aware price display** -- dynamic option prices update correctly
  for incl-tax, excl-tax, and both display modes.
- **Mobile-responsive** -- all option types adapt to small screens.

### Compatibility
- Magento Open Source / Commerce / Cloud 2.4.4 -- 2.4.8
- PHP 8.1, 8.2, 8.3, 8.4
- Hyva Theme 1.1+
